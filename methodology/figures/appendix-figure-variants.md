# Appendix Figure Variants

This appendix provides alternate render variants (compact, wide, rebuttal-ready) for core figures.

## Variant Policy
- `Compact`: optimized for 1-column constraints.
- `Wide`: optimized for 2-column full-width placement.
- `Rebuttal`: same semantics with reduced labels for space.

## Recommended Variants
- F01/F11/F19/F27/F36/F44: provide both compact and wide variants.
- F37/F38/F41: add rebuttal variants emphasizing failure categories.
- F54/F56: include governance-timeline and lineage compact forms.

## Export Checklist
- Verify vector export fidelity.
- Verify grayscale readability.
- Verify caption/axis text is legible in print.
