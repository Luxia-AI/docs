# High-Level Figure Style and Contract (IEEE/ACM)

This directory contains the final publication-facing methodology figure set (`H01`-`H20`).

## Required Figure Contract
Every figure section must include:
1. Figure ID
2. Paper Section
3. Type
4. Research Question
5. Key Variables
6. Mermaid Block
7. Caption (camera-ready)
8. How to Read
9. Expected Insight
10. Failure Signal to Watch
11. Data Source / Log Fields
12. Export Notes

## Rendering and Publication Rules
- Mermaid source is the canonical editable asset.
- Export order: SVG first, PDF second, PNG fallback.
- Grayscale-safe and color-blind-safe design required.
- Annotate each figure as `1-column` or `2-column` recommendation.
- Keep labels concise and deterministic.
